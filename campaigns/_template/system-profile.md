# System Profile — this campaign  (the resolve seam)

> In the v2 engine+companion architecture, L5R task resolution is defined **once** by the
> companion bridge and shared by every L5R campaign:
>
> **→ `.claude/skills/l5r-gm/bridge/system-profile.md`**  (roll-and-keep, the four conflicts, NPC units, routing)
>
> You rarely touch this file. Use it **only** to record a campaign-specific resolution override
> (a house rule, an opt-in subsystem, a special Peril Points pool). If empty, the bridge profile
> stands as-is. Print the active routing with:
> `python3 .claude/skills/mythic-gm/scripts/system.py route`
